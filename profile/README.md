# Criaelo

Plataforma white-label multi-tenant de e-commerce para gráficas e serviços.

Os serviços vivem em repositórios próprios nesta org (extração do monorepo `wilianconte/localgraf` em andamento — ver `criaelo/wiki`). A documentação durável (arquitetura, decisões, backlog, execution-log) é publicada em **wiki.criaelo.com** a partir de `criaelo/wiki`.

## Repositórios

| Repo | Papel |
| --- | --- |
| `scraper` | Microsserviço de scraping de parceiros (folha, sem dependências) |
| `.github` | Workflows reutilizáveis e config da org |
| _(em migração)_ | shop-shared, shop-api, control, control-frontend, shop-admin, shop-frontend, home, wiki |
